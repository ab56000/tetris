# 俄羅斯方塊遊戲實作計畫

這是一份開發純 HTML/JS 俄羅斯方塊遊戲的計畫。目標是打造一個美觀、具備完整功能（包含音效、預覽、多種顏色、旋轉與直接下落）的網頁遊戲。

## User Review Required

請確認以下技術選型與設計方向是否符合您的期望：
> [!IMPORTANT]
> **音效實作方式**：為了避免處理外部音檔下載與跨域問題，我計畫使用瀏覽器內建的 **Web Audio API** 即時合成簡單的 8-bit 風格音效（例如：旋轉音、方塊落地音、消除行音效）。這樣只需要純程式碼即可發出聲音。

## 專案結構

我們將在您的工作目錄（`c:\Users\Lin\.gemini\tetris`）下建立三個主要檔案：
1.  `index.html`：包含遊戲畫布（Canvas）、計分板與「下一個方塊」預覽區。
2.  `style.css`：提供現代化、深色主題（Dark Mode）的視覺設計，讓遊戲看起來有質感。
3.  `script.js`：包含所有的遊戲邏輯與控制。

## 遊戲功能與控制方式

-   **畫布渲染**：使用 HTML5 `<canvas>` 繪製 10x20 網格的遊戲區與 4x4 的預覽區。
-   **方塊 (Tetrominoes)**：實作 7 種標準方塊（I, J, L, O, S, T, Z），每種賦予不同的亮麗顏色。
-   **控制設定**：
    -   `左箭頭 (←)`：向左移動
    -   `右箭頭 (→)`：向右移動
    -   `下箭頭 (↓)`：加速下落 (Soft drop)
    -   `上箭頭 (↑)`：順時針旋轉
    -   `Z 鍵` 或 `Ctrl 鍵`：逆時針旋轉（附加功能，方便左右旋轉）
    -   `空白鍵 (Space)`：直接下落到底 (Hard drop)
-   **系統功能**：
    -   **下一個方塊預覽**：在側邊欄顯示即將出現的方塊。
    -   **計分系統**：消除一行得 100 分，連續消除多行有加成。
    -   **速度提升**：分數越高，方塊下落速度越快。

## 實作步驟

1.  **步驟一：基礎架構與樣式**
    *   建立 `index.html` 與 `style.css`，設定好遊戲容器、Canvas 元素與計分板外觀。
2.  **步驟二：核心邏輯與渲染**
    *   在 `script.js` 建立遊戲矩陣（Board）、方塊資料結構。
    *   實作 Canvas 繪圖函式（畫格子、畫方塊）。
3.  **步驟三：控制與碰撞偵測**
    *   實作鍵盤事件監聽。
    *   加入碰撞偵測（邊界、其他方塊）。
    *   實作移動、旋轉邏輯。
4.  **步驟四：遊戲迴圈與機制**
    *   使用 `requestAnimationFrame` 建立遊戲迴圈。
    *   實作方塊固化、消除整行、計算分數與遊戲結束判定。
    *   實作「下一個方塊」預覽邏輯。
5.  **步驟五：音效與潤飾**
    *   整合 Web Audio API 產生音效。
    *   加入直接下落（Hard drop）功能。
